# apps

Applications and standalone tools built on the MultiBench harness.

- **jaleesbrowser** — application for browsing and exploring MultiBench data.
- **[tradition_validator](tradition_validator/)** — validates that a tradition under
  [`traditions/`](../traditions/) conforms to the canonical file-based format and is set
  up correctly. Python/Typer CLI; see its [README](tradition_validator/README.md) for
  usage. *(Built.)*
